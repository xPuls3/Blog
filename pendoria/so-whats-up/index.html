Angular Rewrite

Everybody knows that Pendoria’s client side code is filled with bugs and lag infested code left over from Pendoria’s initial launch just under 2 years ago. As a result Xortorx has decided it is time to clean up said code and ensure it runs both smoother and faster. As Xotrox mentioned in the latest news post that his current goal is to rewrite all of the current client side code inside Angular 8+. While this doesn’t mean anything much to regular players, it will mean less bugs, less lag, and and an overall better experience.

TLDR; Less bugs and better performance in the future!

​

But… Content?

As Xortox stated in the news post I mentioned earlier, due to the development team focusing mostly on the rewrite I doubt anyone will be able to promise any new content any time soon. Once we feel fairly comfortable with the rewrite he’ll most likely evaluate the possibility of working on some new content on the side.

The content that was already on the ToDo list is still there, it just got knocked behind the rewrite. On the upside we should be seeing less development bumps in the future.
